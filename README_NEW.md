# Open Data Governance Schema (ODGS)

[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)
[![Status](https://img.shields.io/badge/Status-Standard-success)]()

**The Universal Standard for Headless Data Governance.**

ODGS is an open protocol that defines metrics, lineage, and data quality rules in a vendor-neutral format (JSON). It serves as the single source of truth for **AI Agents**, **BI Dashboards**, and **Data Platforms**.
